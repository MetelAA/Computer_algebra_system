# Система компьютерной алгебры
### Цель:
Целью стала разработка системы компьютерной алгебры с поддержкой длинной арифметики для натуральных, целых и рациональных чисел, а также полиномов с натуральными коэффициентами.
### Результат:
Результатом стала программа на языке C++ с поддержкой всех необходимых операций и множеств. Конечная программа представлена вместе с визуальном интерфейсом. Программа поддерживает запуск на ОС Linux и Windows, для скачивания и использования смотрите раздел [Releases](https://github.com/MetelAA/DMaTGColloquium/releases)
